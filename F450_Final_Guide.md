## 第一部分：全方位地面调试流程 (Pre-Flight Check)

**目标**：在不装螺旋桨的情况下，验证 传感器方向、遥控逻辑、电机转向、PID 修正方向 四大核心要素。

### 准备工作
1.  **拆下所有螺旋桨！拆下所有螺旋桨！拆下所有螺旋桨！** (重要的事情说三遍)
2.  将飞控板暂时固定在机架中心 (可以用皮筋或双面胶临时固定)，**一定要确保传感器 X 轴箭头指向机头 (红色臂方向)**。
3.  连接电池，飞控上电。如果使用蓝牙调参，连接手机 APP；如果用电脑，连接 USB 线并打开串口助手 (9600 baud)。

### 步骤 1：静态数据检查 (心跳检测)
*   **观察串口数据**：
    *   `R` (Roll): 左右晃动飞机，数值是否有变化？
    *   `P` (Pitch): 前后晃动飞机，数值是否有变化？
    *   `Y` (Yaw): 水平转动飞机，数值是否有变化？
*   **验证零点**：
    *   把飞机放在水平地面上，静止 3 秒。
    *   `R` 和 `P` 应接近 **0** (允许 ±3 度误差)。如果不为 0，请重启飞控让其自动校准。

### 步骤 2：传感器方向验证 (最为关键！)
*这个反了必炸机！*

| 动作 | 预期 R (Roll) | 预期 P (Pitch) | 状态 |
| :--- | :--- | :--- | :--- |
| **右翼抬高** (机身左压) | **变小 (负数)** | 无明显变化 | [ ] 通过 |
| **左翼抬高** (机身右压) | **变大 (正数)** | 无明显变化 | [ ] 通过 |
| **机头抬高** (屁股压) | 无明显变化 | **变大 (正数)** | [ ] 通过 |
| **机头低下** (头压) | 无明显变化 | **变小 (负数)** | [ ] 通过 |

**之前你测试的结果 (已确认正确)：**
- 机头抬高 → P 为正 ✓
- 右边抬高 → R 为负 ✓

### 步骤 3：遥控器逻辑验证
*验证你的摇杆是不是反了。*

1.  **解锁飞机**：左摇杆油门最低 + 偏航最右（外八字），保持 2 秒。
    *   即：左摇杆推到右下角，保持 2 秒。
2.  推一点点油门 (超过 5%)，让电机开始转动。
3.  **Roll (横滚) 测试** - 右摇杆左右：
    *   摇杆 **向右打** → 串口显示的目标角度应为 **正**。
    *   摇杆 **向左打** → 目标角度应为 **负**。
4.  **Pitch (俯仰) 测试** - 右摇杆前后：
    *   摇杆 **向前推 (低头)** → 目标角度应为 **负**。
    *   摇杆 **向后拉 (抬头)** → 目标角度应为 **正**。
5.  **Yaw (偏航) 测试** - 左摇杆左右：
    *   摇杆 **向左打** → 目标角速度应为 **负** (逆时针)。
    *   摇杆 **向右打** → 目标角速度应为 **正** (顺时针)。

### 步骤 4：电机转向验证
*用手指轻碰电机外壳感受震动方向，或者用纸条碰桨轴。*

保持解锁，推一点油门让电机慢转：

| 电机 | 位置 | 正确旋转方向 | 验证方法 | 状态 |
|------|------|--------------|----------|------|
| M1 | 右前 (PA8) | **CCW 逆时针** | 从上往下看，桨叶逆时针转 | [ ] 通过 |
| M2 | 左前 (PA9) | **CW 顺时针** | 从上往下看，桨叶顺时针转 | [ ] 通过 |
| M3 | 左后 (PA10) | **CCW 逆时针** | 从上往下看，桨叶逆时针转 | [ ] 通过 |
| M4 | 右后 (PB6) | **CW 顺时针** | 从上往下看，桨叶顺时针转 | [ ] 通过 |

**错误修正**：如果某个电机方向反了，交换该电调连接电机的**任意两根线** (黄/蓝/红三根线，换两根即可反向)。

### 步骤 5：PID 修正方向验证 (保命测试 - Hand Test)
*这是最后一道防线。确保飞控是在"救你"而不是"害你"。*

**保持解锁，油门 10-15%，用手牢牢抓紧机身底部 (注意安全！)：**

1.  **猛烈向前压机头 (模拟低头)**：
    *   **预期**：**前面的电机 (M1右前, M2左前)** 瞬间加速尖叫，试图把头抬起来。
    *   ❌ 如果后面的电机加速 → Pitch 方向反了 → **必炸**。

2.  **猛烈向右压机身 (模拟右倾)**：
    *   **预期**：**右边的电机 (M1右前, M4右后)** 瞬间加速，试图把右边顶回去。
    *   ❌ 如果左边的电机加速 → Roll 方向反了 → **必翻**。

3.  **猛烈逆时针转机身 (向左急转)**：
    *   **预期**：**CCW电机 (M1右前, M3左后)** 瞬间加速，产生向右(顺时针)的反扭力来抵抗。
    *   *(Yaw 比较难感觉，只要前两步对，通常问题不大)*

## 第二部分：PCB 画板指南 (底板/转接板)

**设计目标**：适配 STM32G431 核心板 + F450 机架接口。

### 1. 原理图设计检查表
*   [ ] **核心板接口**：使用 2排 20Pin (或其他规格) 的排母。确保引脚定义和你的蓝色板子丝印一一对应（特别是 5V, GND, PA0-PA15, PB6）。
*   [ ] **电源模块**：
    *   输入：焊盘或 XT60 接口 (接 3S 电池)。
    *   降压：建议板载一个 5V 降压模块 (如 Mini360 模块封装)，或者留出 5V 输入针脚 (从电调取电)。
    *   **注意**：如果你从电调取电，**只连接 M1 的红线到 VCC**，M2/M3/M4 的红线悬空 (NC)。
*   [ ] **电机接口 (ESC)**：
    *   4组 3Pin 排针 (GND, 5V, Signal)。
    *   **M1 (右前)**: GND, 5V(可连VCC), PA8
    *   **M2 (左前)**: GND, NC, PA9
    *   **M3 (左后)**: GND, NC, PA10
    *   **M4 (右后)**: GND, NC, PB6 (注意是TIM4)
*   [ ] **接收机接口**：GND, 5V, PA0 (TIM2_CH1 PPM)。
*   [ ] **IMU 接口 (ICM42688)**：
    *   SPI 接口 (GND, 3.3V, SCK, MISO, MOSI, CS)。
    *   **强烈建议**：不要焊死在板上，留出 2.54mm 排孔，用软排线/杜邦线把传感器模块引出来，以便进行**减震安装**。如果直接焊死在底板上，F450 的震动会让你的飞机在空中抽搐。
*   [ ] **蓝牙接口 (HC-05/06)**：GND, 5V, TX→PA3(RX2), RX→PA2(TX2)。
*   [ ] **电池电压检测**：
    *   做一个简单的分压电路：电池正极 → R1(10kΩ) → PA1 → R2(1.5kΩ) → GND。
    *   最大测量电压 = 3.3V × (10k+1.5k) / 1.5k ≈ 25V，足够 3S-4S 使用。

### 2. PCB Layout (画板) 注意事项
*   [ ] **线宽**：信号线 10-15mil 即可；**电源线 (5V, GND, BAT)** 尽量粗，至少 20-30mil，或者直接铺铜。
*   [ ] **铺铜 (Pour)**：**顶层和底层都要铺 GND 铜**，并打满过孔 (Via) 连接两层地。这对于抗干扰至关重要。
*   [ ] **布局**：
    *   核心板放中间或一边。
    *   电机插针放在板子四个角，方便走线。
    *   接收机放尾部。
    *   **IMU 安装位**：如果非要板载，必须放在**几何中心**，且远离蜂鸣器和电流线路。
*   [ ] **丝印**：一定要在板子上标清楚 **机头方向 (▲箭头)**，以及 **M1, M2, M3, M4** 的位置，防止接错线。

### 3. 推荐的"防炸"小设计
*   **加一个蜂鸣器 (有源)**：接在 PC13 (LED口) 或者单独找个 GPIO 通过三极管驱动。低压报警、找飞机都很有用。
*   **预留 I2C 接口**：以后万一你想加磁力计或气压计定高，留个 I2C (PB7/PB8) 接口会很方便。

## 第三部分：明日调试流程总结

按顺序做完以下步骤，全部打勾后即可装桨试飞：

- [ ] 1. 上电，串口显示 `R:0 P:0` (水平校准正常)
- [ ] 2. 传感器方向：抬头P正，右压R负
- [ ] 3. 电机转向：M1/M3 逆时针，M2/M4 顺时针
- [ ] 4. 遥控器响应：打杆方向与飞机倾斜方向一致
- [ ] 5. PID Hand Test：压哪边，哪边电机反抗
- [ ] 6. 零油门停转功能正常
- [ ] 7. 解锁/锁定功能正常

**全部通过后，装上正反桨 (有字朝上)，找个空旷草地，准备起飞！**
