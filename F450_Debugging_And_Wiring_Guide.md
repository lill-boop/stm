# F450 飞控调试与接线完全指南

这份指南将指导你如何将 STM32G431 自制飞控与 F450 机架硬件进行连接和调试。请**严格按照步骤操作**，安全第一！

---

## ⚠️ 核心安全警告 (必读)

1.  **绝对禁止装桨调试**：在所有电机测试完成、确认方向正确且控制逻辑正常之前，**严禁安装螺旋桨**。此时的电机转速极快，螺旋桨如飞刀般危险。
2.  **防止电源短路**：接线时务必断电。确保所有焊点无连锡，正负极绝不接反。
3.  **电压匹配**：接收机信号线 output (5V) 到 STM32 输入 (3.3V) 必须串联电阻保护。电调 BEC (5V) 不要和 USB 同时长时间供电。

---

## 🛠️ 第一部分：硬件接线

### 1.1 电机顺序与布局 (非常重要)

根据你的代码逻辑 (`motor.c`)，电机布局如下 (X 型四轴)：

```
      机头 (前方)
         ▲
    M2 (左前)   M1 (右前)
(PA9, CW)       (PA8, CCW)
         \     /
          \   /
           \ /   <-- 飞控板
           / \
          /   \
         /     \
    M3 (左后)   M4 (右后)
(PA10, CCW)     (PA11, CW)
```

**连接表：**

| 电机位置 | STM32引脚 | 旋转方向(从上往下看) | 电调信号线(细白/黄线) |
| :--- | :--- | :--- | :--- |
| **M1** (右前) | **PA8** | **CCW (逆时针)** | 接 PA8 |
| **M2** (左前) | **PA9** | **CW (顺时针)** | 接 PA9 |
| **M3** (左后) | **PA10** | **CCW (逆时针)** | 接 PA10 |
| **M4** (右后) | **PA11** | **CW (顺时针)** | 接 PA11 |

*注：若电机旋转方向反了，**任意交换该电机与电调连接的三根粗线中的两根**即可反向。*

### 1.2 接收机 (FS-iA6B) 连接

接收机使用 PPM 模式输出（请确保你的遥控器设置已开启 PPM 输出）。

| 接收机引脚 | 功能 | 连接到飞控 | 注意 |
| :--- | :--- | :--- | :--- |
| **PPM/CH1** | 信号 | **PA0** | **强烈建议串联 1kΩ 电阻保护 PA0** |
| **VCC** | 5V电源 | 5V (来自电调BEC) | 接收机需要 5V 供电 |
| **GND** | 地 | GND | 必须共地 |

### 1.3 电池与电源连接

*   **动力电**：电池 XT60 -> F450 电路板 -> 4个电调的红黑粗线。
*   **飞控供电**：任选一个电调的细红线(5V) 和细黑线(GND) 接到飞控的 5V 和 GND。
    *   (注意：只需接那一根红线供电即可，其他3个电调的红线可以挑出来包好，避免并联供电隐患，但F450套装电调通常是线性BEC，并联也不怕)。
*   **电压检测**：
    *   将电池的正极 (+) 接到分压电阻 (30k) 的一端。
    *   电阻另一端接到 **PA1**。
    *   PA1 还需要接一个 10k 电阻到 GND。
    *   (即: 电池+ -> 30k -> PA1 -> 10k -> GND)。

### 1.4 IMU (ICM42688) 连接

| IMU 引脚 | 功能 | 飞控引脚 |
| :--- | :--- | :--- |
| VCC | 3.3V | 3.3V |
| GND | 地 | GND |
| SCL/SCK | 时钟 | PA5 |
| SDA/SDI | MOSI | PA7 |
| AD0/SDO | MISO | PA6 |
| NCS/CS | 片选 | **PB0** |

## 💻 第二部分：调试步骤

### 2.1 接收机测试 (不接动力电，只插 USB)

1.  将飞控通过 USB 连接电脑。
2.  打开串口监视器 (波特率 921600)。
3.  打开遥控器。
4.  此时串口应显示 `RC` 字样，后面跟着 `T:xxxx` (油门值)。
5.  输入 CLI 命令 `STATUS` 或观察打印数据：
    *   推油门，`T` 值应增大 (1000 -> 2000)。
    *   摇杆打左，`R` (Roll) 应变小；打右应变大。
    *   摇杆推前，`P` (Pitch) 应变大；拉后应变小。
    *   **如果方向反了**，在遥控器菜单里设置通道反向 (Reverse)。

### 2.2 IMU 姿态测试 (只插 USB)

1.  拿起飞控板，保持机头朝前。
2.  **向右倾斜**：串口显示的 `R` (Roll) 应该是正数 (比如 +30)。
3.  **机头抬起**：串口显示的 `P` (Pitch) 应该是正数。
4.  **逆时针转**：串口显示的 `Y` (Yaw) 应该变化。
5.  如果数据对不上，说明 IMU 安装方向有误，需要在代码里修改 `IMU_ALIGN` (这需要改代码，暂时默认你的安装方向是正确的)。

### 2.3 电机解锁与转向测试 (接电池，**严禁装桨**)

1.  断开 USB。
2.  插上电池 (此时电调会自检 "滴-滴-滴~")。
3.  **解锁**：(根据你的代码逻辑)
    *   油门拉到最低。
    *   同时将方向摇杆 (Yaw) 打到最左或最右保持 2 秒。
    *   观察串口 (如能接TTL) 或听电机声音。
4.  **轻推油门**：
    *   所有电机应开始缓慢转动。
5.  **检查转向**：
    *   用手轻触电机侧面，确认 M1/M3 是逆时针，M2/M4 是顺时针。
    *   **如果反了**：交换该电机任意两根粗线。
6.  **检查对应的电机位置**：
    *   向右打滚转 (Roll Right)：
        *   机身右侧 (M1, M4) 应该减速/停转。
        *   机身左侧 (M2, M3) 应该加速。
        *   (这是最直观的验证“电机顺序有没有接错”的方法)。

## ✈️ 第三部分：PID 调参 (CLI 命令)

你可以通过串口发送命令来修改参数 (参数会保存在 Flash 中)。

*   `HELP` : 显示帮助
*   `Saved PID` : 保存当前参数
*   `SET P ROLL 1.5` : 设置横滚 P 值为 1.5
*   `SET I PITCH 0.05` : 设置俯仰 I 值为 0.05
*   `SHOW` : 显示当前所有参数

**默认 PID 可能比较保守，第一次试飞建议：**
1.  找个开阔草地。
2.  起飞时果断推油门，不要在地面长时间磨蹭 (地面震动会干扰 I 项)。
3.  如果飞机高频抖动 (金蝉鸣叫)，说明 P 太高，降低 P。
4.  如果飞机反应迟钝、飘，说明 P 太低，增加 P。
